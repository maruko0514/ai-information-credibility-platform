關於本專題

AI 資訊泡泡與認知偏誤分析平台


研究背景

隨著社群媒體、生成式 AI 與網路平台快速發展， 使用者每天都會接觸大量資訊，但這些資訊可能受到 演算法推薦、單一來源及情緒化內容影響， 容易形成資訊泡泡（Information Bubble）與認知偏誤（Cognitive Bias）。

本專題希望透過 AI 分析、網頁爬蟲及資料分析技術， 協助使用者從多個角度評估網路資訊， 提升媒體識讀能力與資訊判斷能力。


研究目的

  • 分析網站可信度
  • 評估資訊泡泡風險
  • 偵測認知偏誤
  • 分析多元觀點程度
  • 提供 AI 改善建議
  • 建立資訊健康度（Information Health Index）

系統分析流程

  1. 使用者輸入網址
  2. 網站爬蟲擷取內容
  3. 分析網站可信度
  4. 分析資訊泡泡風險
  5. 分析認知偏誤
  6. 分析多元觀點
  7. 計算資訊健康度（IHI）
  8. AI 提供分析摘要與建議
  9. 儲存至 Dashboard 與歷史紀錄

使用技術

  • Python
  • Flask
  • BeautifulSoup
  • Requests
  • SQLite / PostgreSQL
  • HTML / CSS
  • JavaScript
  • Matplotlib
  • AI 文字分析
  • Render 雲端部署

專題特色

  • ✅ 網站可信度分析
  • ✅ 資訊泡泡分析
  • ✅ 認知偏誤分析
  • ✅ 多元觀點分析
  • ✅ AI 綜合建議
  • ✅ Information Health Index（資訊健康度）
  • ✅ Dashboard 統計分析
  • ✅ 歷史紀錄管理
